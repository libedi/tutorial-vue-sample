Vue 인스턴스

  • Vue 인스턴스: Vue 함수로 생성 => var vm = new Vue( { ... } )
  • 속성과 메소드:
    1. Vue 인스턴스는 data 객체에 있는 모든 속성을 프록시처리 => reference 와 유사한 의미
    2. data에 있는 속성들은 인스턴스가 생성될 때 존재한 것들만 반응형! => 이미 생성된 인스턴스에 새 속성을 추가하면 그 데이터가 변해도 화면이 갱신되지 않음.
    3. 따라서, 반응형이 필요한 데이터 항목은 인스턴스 생성시 기본값으로 추가되어야 함. 
      data: {
    newTodoText: '',
    visitCount: 0,
    hideCompletedTodos: false,
    todos: [],
    error: null
}
    4. Object.freeze() 는 기존 속성이 변경되는 것을 막음 => 반응성 시스템이 추적하지 못하게 함.

      {{ foo }}
